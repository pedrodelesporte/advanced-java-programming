## Advanced Java Programming

This repository contains my coursework and projects for Advanced Java. Through this curriculum, I have mastered sophisticated features of the Java programming language and modern development workflows.

### Core Competencies Developed
* **Software Engineering Tools:** Building and managing Java projects using the **Maven** build tool and implementing automated unit testing with **JUnit**.
* **Object-Oriented Mastery:** Applying advanced OOP concepts including class abstraction, inheritance, polymorphism, and dynamic binding to develop modular software.
* **Concurrent Programming:** Designing and implementing **multithreaded** applications to handle complex, simultaneous tasks.
* **Data & Networking:** Accessing and updating databases via **JDBC and JPA**, and building applications that utilize **Network I/O**.
* **Web Development:** Developing server-side logic using **Java Servlets** (running in Tomcat) and building full-stack applications with **Spring and Spring Boot**.
